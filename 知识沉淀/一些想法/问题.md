# 电脑问题排查
1. 电脑自动重启：打开 事件查看器->Windows日志->系统。看看Kernel-Power（事件 ID 41）。如果有这个事件，那说明应该是硬件断电，可能的原因如下：
**电源（PSU）问题**
这是最像的。
**典型表现**：
- 平时用没事
- 一进游戏 / 玩一会 → 功耗拉高 → 直接重启
- 不蓝屏、不报错、不冻结
**可能原因**：
- 电源额定功率不够
- 老化（电容虚了）
- 显卡瞬时功耗峰值触发保护
- 杂牌 / 山寨电源
**显卡过热或显卡供电异常**
尤其是：
- 风扇灰多
- 显存温度高
- 显卡双 8pin / 12VHPWR 接触不好
过热时，显卡会：
- 先掉驱动（你没提）
- 再不行 → **主板直接断电保护**
**CPU 过热 / 主板供电（VRM）**
- CPU 温度瞬间 95℃+
- 主板供电扛不住
- BIOS 设置过激（PBO / 自动超频）
解决方案：
把电脑电池调成性能模式就可以了。
**把最大处理器状态设为 99%**
路径：
> 控制面板 → 电源选项 → 当前计划 → 更改高级电源设置  
> → 处理器电源管理  
> → 最大处理器状态 = **99%**
作用只有一个：
> **彻底禁用 Turbo 峰值**
